# Energy System Modelling

A portfolio of energy system modelling projects independently completed by working through the open course materials from [*Data Science for Energy System Modelling*](https://github.com/fneum/data-science-for-esm) at TU Berlin by [Fabian Neumann](https://github.com/fneum).

## Attribution

This repository contains my independent solutions, modelling work, and analysis — completed outside of TU Berlin as self-directed study.

Raw data files are not included; download links are provided in each notebook.
